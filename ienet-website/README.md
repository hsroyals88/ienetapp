# IeNet - Professional IT Services Website

A comprehensive website for IeNet, showcasing professional IT services including web development, cybersecurity, hosting, digital marketing, and more. Built with modern web technologies and optimized for performance and SEO.

## 🌟 Features

- **Comprehensive Service Portfolio**: 12 main service categories with detailed sub-services and feature pages
- **Responsive Design**: Mobile-first approach with modern CSS Grid and Flexbox
- **SEO Optimized**: Hierarchical URL structure, meta tags, schema markup, and breadcrumb navigation
- **Professional UI/UX**: Clean, modern design with interactive elements and animations
- **Hierarchical Structure**: Main Service → Sub-Service → Feature Detailed pages
- **Cross-browser Compatible**: Works seamlessly across all modern browsers

## 🚀 Getting Started

### Prerequisites
- Web server with PHP 7.4+ support (Apache/Nginx)
- Modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ienet-website.git
   cd ienet-website
   ```

2. **Set up web server**
   - For local development, use XAMPP, WAMP, or MAMP
   - Point document root to the project directory
   - Ensure PHP is enabled

3. **Access the website**
   - Open `http://localhost/ienet-website` in your browser
   - Navigate through the hierarchical service structure

### Development Setup

1. **Local Server**
   ```bash
   # Using PHP built-in server
   php -S localhost:8000
   ```

2. **File Structure**
   - All PHP files use relative paths for includes
   - CSS and JS files are in the assets directory
   - Images should be placed in assets/images/
